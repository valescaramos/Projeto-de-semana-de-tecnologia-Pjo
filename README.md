#Projeto-de-semana-de-tecnologia-Pjo


Este projeto é um site criado para a disciplina de Desenvolvimento Web.
O objetivo é apresentar informações sobre a Semana Olímpica de Tecnologia, contendo páginas sobre palestras, oficinas, inscrições e detalhes do evento.



 Tecnologias Utilizadas

* HTML5
* CSS3 (arquivo próprio)
* Bootstrap 5
* Git e GitHub



 Estrutura do Projeto

 Responsividade

O site foi desenvolvido utilizando o grid do Bootstrap, garantindo que funcione corretamente em celulares, tablets e computadores.



Personalização

O projeto possui CSS próprio para cores, fontes, espaçamentos e ajustes visuais que complementam o Bootstrap.

Apresentar um site organizado, funcional e visualmente agradável sobre um evento tecnológico, atendendo aos critérios de estrutura, estilo, responsividade e publicação.
